### 💼 O que eu faço na prática (Destaques da Experiência)

Atualmente como **Analista de Planejamento Júnior**, atuo na ponta de ponta a ponta (End-to-End) do fluxo de dados:
- **Engenharia & ETL:** Criação e estruturação automatizada de tabelas no SQL Server, garantindo performance e fontes da verdade confiáveis.
- **Automação de Reportes:** Desenvolvimento de rotinas (*Jobs* no SQL Server) para processamento direto no banco e exportação/envio automático de relatórios via e-mail (zerando tarefas manuais).
- **Otimização de Processos (Pushdown):** Preparação e limpeza de bases diretamente no Microsoft SQL Server para alimentar ferramentas de visualização de forma leve.
- **Análise & Data Viz:** Criação de relatórios executivos no Power BI com foco avançado em interface de usuário (UI) e métricas de negócio para suporte à diretoria.
- **Governança:** Gestão de rotinas de transferência segura de dados via ambiente SFTP.
